DJATIGUI

La procédure d'installation Djatigui.

1. Rassurez vous que vous avez une connexion internet.

2. Cliquer sur le fichier téléchargé.

3. Cliquer sur : Télécharger.

4. Cliquer sur : Installer quand même.

5. Cliquer sur : Ne pas analiser.

6. Cliquer sur : Ouvrir.

7. Assurez vous que vous avez une connexion internet et choisir une ville.

8. Une fois la ville choisie, Veuillez cliquer sur: Entrer

9. Si tout se passe bien, vous devez avoir la page d'accueil.
